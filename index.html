Detailed View

Full course information: the lecturer’s name, classroom and building, complete subject title, order of classes, and type of activity.

Open-Source

Users can review and modify as needed. Active feedback and contributions keep the project reliable.

Automated Up-to-Date

Scripts for scraping, parsing, fetching, and saving are run every day so schedules always stay current.

Get Your Calendar

<<Instruction>>

Navigate to Schedules selecting one of available buttons or links on website.

Select your faculty from available options.

Once redirected to your faculty, select your group from the list.

After selecting your group, add the schedule by clicking apple "Subscribe Calendar" button.

(Optional) If nothing happens, you can add it manually by copying the link under the subscribe button and paste it in:
Calendar > Calendars > Add calendar > Add subscription calendar.

Confirm the subscription in the pop-up window by clicking "Find".

Customize your calendar settings as needed and click confirm button at the right-top.

You're all set! Your WATcalendar will now appear in the Apple Calendar app with all the latest schedule information. Enjoy!

Navigate to Schedules selecting one of available buttons or links on website.

Select your faculty from available options.

Once redirected to your faculty, select your group from the list.

After selecting your group, copy the link above the Google button starting with https://
and redirect to Google Calendar Settings by hitting the button.

Log in to your Google account.

On Desktop you need to go for "+" in "Other calendars" section and select "From URL" like on the screenshot.

You will be redirected to Google Calendar subscription Settings.

!IMPORTANT
Paste the link into the "URL" field and select "Set calendar as public"

Once you subscribe the calendar, leave the Settings section and after few seconds you will see your schedule.

You're all set! Your WATcalendar will now appear in the Google Calendar with all the latest schedule information. Enjoy!

Manual Installation

You can install the WATcalendars project available on Github that include Python Scripts.

Clone the repository to your local machine.

git clone https://github.com/dominikx2002/WATcalendars.git sh

Next add rights to the setup script.

cd WATcalendars && chmod +x setup.sh sh

Running following script will automaticaly install required dependencies.

./setup.sh sh

After that, you can Use the scraping scripts, Change them, Customize them, Fix as needed.

In the repository check "help.txt" file for available options and useful informations or simply type cat help.txt:

cat help.txt sh

Setup script to install the required dependencies are included for Debian, Ubuntu, or Arch.
For other distros, please install the dependencies first.
